## Maps Header file for C++
* ## __Constructor__
  ##### ```Maps<keys_variable_type, values_variable_type>```
  creates a map object
  
* ## __insert__
  ##### ```insert(key, value)```
  pushes a key value pair into the map
  
* ## __remove__
  ##### ```remove(key, value)```
  removes the given key and the corresponding value from the map;
